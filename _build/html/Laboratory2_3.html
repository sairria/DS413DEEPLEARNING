import numpy as np

#1 Define Activation Functions & their derivatives

def relu(z):
    """ReLU activation function: max(0, z)"""
    return np.maximum(0, z)

def relu_derivative(z):
    """Derivative of ReLU: 1 if z > 0, 0 otherwise"""
    # Create a copy of z and set values based on the condition
    dz = np.where(z > 0, 1, 0)
    return dz

#3 Forward Propagation 

print("--- FORWARD PASS ---")

# Step 1: Calculate the weighted input for the hidden layer (Z1)
# Z1 = W1^T * x + theta_1. Since W1 is 3x2 and x is 3x1, we need to transpose W1.Alternatively, use W1 (3x2) and transpose x (1x3), then transpose the result. A cleaner way is Z1 = x^T @ W1 + theta_1^T, then transpose Z1 for a 2x1 vector. Let's stick to the convention: Z1 = W1_T @ x + b1. Here, W1 is (Input_Size x Hidden_Size). We assume x is a 3x1 vector. Z1 will be (Hidden_Size x 1) = (2x1)
Z1 = np.dot(W1.T, x) + theta_1
# Z1 = [[0.2 * 1 + 0.4 * 0 + (-0.5) * 1 - 0.4], [-0.3 * 1 + 0.1 * 0 + 0.2 * 1 + 0.2]]
print(f"Z1 (Hidden Weighted Input):\n{Z1}")
# Expected Z1: [[-0.7], [0.1]]

# Step 2: Apply ReLU activation to get hidden layer output (A1)
A1 = relu(Z1)
print(f"A1 (Hidden Activation):\n{A1}")
# Expected A1: [[0.0], [0.1]]

# Step 3: Calculate the weighted input for the output layer (Z2)
# Z2 = W2^T * A1 + theta_2. W2 is 2x1, A1 is 2x1. Z2 will be (Output_Size x 1) = (1x1)
Z2 = np.dot(W2.T, A1) + theta_2
# Z2 = [[-0.3 * 0.0 + (-0.2) * 0.1 + 0.1]]
print(f"Z2 (Output Weighted Input):\n{Z2}")
# Expected Z2: [[0.08]

KEY TEAKEAWAYS#

Zero-Out Effect of ReLU: Since the first hidden unit’s weighted input, Z 1 ​ [0], was −0.7 (negative), its activation A 1 ​ [0] became 0 due to the ReLU function.

Backpropagation Stop: Because A 1 ​ [0] is 0, the weight W 2 ​ [0] (the first output weight) is effectively ignored in the forward pass. More critically, the gradient for the first hidden unit’s weights (W 1 ​ column 1) is zero due to the derivative of ReLU being zero for negative Z 1 ​ . This means none of the weights feeding into the first hidden unit will be updated in this step.

Weight Updates: Only the weights and biases associated with the second hidden unit (which had a positive Z 1 ​ ) and the output layer are updated.
